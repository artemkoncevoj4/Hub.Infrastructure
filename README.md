## Стек технологий
| Слой | Технология | Роль в проекте |
| ---- | ---------- | -------------- |
| Frontend | Vue 3 | Единый интерфейс, общающийся с API |
| Gateway | Nginx | Единая точка входа. Принимает запросы от UI и раскидывает их по нужным микросервисам (Reverse Proxy) |
| Backend | C#, ASP.NET Core 9 | Написание самих микросервисов |
| Базы данных | PostgreSQL | Основное хранилище (у каждого сервиса своя логическая БД) |
| Кеш & Сессии | Redis | Хранение Refresh-токенов и кэширование частых запросов |
| Брокер сообщений | Kafka | Ассинхронная шина данных для общения сервисов |
| Интеграция брокера | MassTransit | C#-фреймворк, чтобы не писать низкоуровненый код для Kafka |
| Инфраструктура | Docker + Compose | Изоляция и запуск всего проекта |

## Структура файлов
```Plaintext
Hub/
├── .git/
├── docker-compose.yml          # Поднимает БД, Kafka, Redis, Nginx и сами сервисы
├── nginx/
│   └── nginx.conf              # Правила маршрутизации (кто куда идет)
├── src/
│   ├── BuildingBlocks/
│   │   └── Hub.Shared/         # Общие DTO и интерфейсы (PingEvent), собирается в NuGet или подключается ссылкой
│   ├── Services/
│   │   ├── Hub.Identity.API/   # Микросервис авторизации
│   │   └── Hub.VehicleLog.API/ # Микросервис логов авто
│   └── Frontend/
│       └── hub-ui/             # Исходники Vue/Svelte
└── Hub.sln                     # Общий Solution для удобного открытия бэкенда в IDE
```
